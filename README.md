# 🌱 KhetHub Smart Agriculture Dashboard

## 📌 Project Overview

KhetHub Smart Agriculture Dashboard is a modern web-based dashboard designed to help farmers monitor and manage their farming activities efficiently. The dashboard provides real-time agricultural insights through interactive charts, KPI cards, weather information, soil monitoring, crop health analysis, irrigation tracking, alerts, and AI-based recommendations.

This project was developed as part of the KhetHub Dashboard Challenge to demonstrate frontend development, UI/UX design, data visualization, and dashboard development skills.

## 🚀 How to Run

1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Install the Live Server extension.
4. Right-click on `index.html`.
5. Click **Open with Live Server**.

The dashboard will open in your browser.
